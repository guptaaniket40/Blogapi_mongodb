# 📝 Blog API (Django + MongoDB)

A simple and scalable Blog API built using **Django Rest Framework** and **MongoDB (MongoEngine)**.  
This API allows users to create, read, update, and delete blog posts.

---

## 🚀 Features

- Create Blog Post
- Get All Posts
- Get Single Post
- Update Post
- Delete Post
- MongoDB Integration (No SQL)
- RESTful API Structure

---

## 🛠️ Tech Stack

- Python
- Django
- Django Rest Framework
- MongoDB
- MongoEngine
